## How to Run This Programs
1. Install XAMPP
2. Clone repository ini ke dalam folder di dalam folder xampp/htdocs
3. Buat database MySQL dan jalankan script init.sql
4. Ubah variable `$dbName` di file `connection.php` sesuai dengan nama database yang telah dibuat
5. Jalankan `home.php` di browser.
